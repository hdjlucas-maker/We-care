# 🌐 Projeto Responsivo - DevClub

Este projeto foi desenvolvido durante meus estudos no **DevClub**, aplicando **HTML e CSS** para criar uma página totalmente **responsiva**, compatível com **desktop, tablet e mobile**.  

Foi um grande passo na minha jornada, pois consegui entender melhor como estruturar layouts que se adaptam a diferentes dispositivos, garantindo uma experiência consistente para o usuário. 🚀

## 🎯 Objetivo
- Criar uma página web responsiva
- Aplicar boas práticas de HTML e CSS
- Garantir compatibilidade em múltiplos dispositivos

## 🛠️ Tecnologias utilizadas
- **HTML5**
- **CSS3**

Esse projeto me mostrou que **responsividade é essencial** no desenvolvimento web moderno. Cada ajuste feito para adaptar a página em diferentes dispositivos reforçou minha confiança e me aproximou ainda mais do meu objetivo de atuar como desenvolvedor front-end. 💻📱
